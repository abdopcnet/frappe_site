# Technology Stack

## Frontend

- **Vue.js 3** - Progressive JavaScript framework
- **Vue Router** - Client-side routing
- **mitt** - Event bus for component communication
- **Pure CSS** - No UI frameworks (no Vuetify)

## Backend

- **Python 3.10+** - Programming language
- **Frappe Framework v15** - Web framework
- **ERPNext v15** - ERP system integration

## Build Tools

- **Frappe Build System** - esbuild for bundling
- **Yarn** - Package manager

## Database

- **MySQL/MariaDB** - Database system
